# Nav

To display the navigation links.

## Examples

> Use `disabled` and `active` props to highlight or disable links.

## Usage

`import { Nav, NavLink } from "ui-neu"`

```jsx
<Nav>
  <NavLink href="" key="1">NavLink1</NavLink>
  <NavLink href="" key="2">NavLink2</NavLink>
</Nav>
```
